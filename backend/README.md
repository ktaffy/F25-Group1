# Backend

This is the backend service for the Mise en Place cooking companion app. It is built with **Node.js**, **Express**, and **TypeScript**, with API documentation provided via **Swagger**.

## Requirements
- Node.js v18+
- npm or yarn
- A `.env` file (see `.env.example` if provided)

## Installation
```bash
# Install dependencies
npm install
```

## Development
```bash
# Run in development mode (using tsx)
npm run dev
```

## Build and Start
```bash
# Compile TypeScript
npm run build

# Run compiled code
npm start
```

## Linting
```bash
npm run lint
```

## API Documentation
### Swagger UI is available after starting the server:
```bash
http://localhost:3000/api-docs
```
This provides an interactive interface to explore and test the backend API.
